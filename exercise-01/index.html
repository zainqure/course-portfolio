Exercise 01: Create and Deploy Your Course Portfolio

Prompt

Set up your professional development environment, create the repository for your Course Portfolio, and deploy a simple Course Homepage to the live internet using GitHub Pages.

Detailed Explanation

This single, comprehensive exercise requires you to run through the entire setup and deployment process. By the end, you will have a live, public URL that will serve as the central homepage for all your coursework this semester.

Exercise Instructions

  • Read "Setting up Your Course Portfolio" in the Week 02 module. Follow the instructions it provides.
  • Edit your Homepage File: Inside your cloned repository folder, open the index.html file, and edit it using VS Code. Change the placeholder text with your information, as you prefer. Save your file changes when finished.
  • Commit and Push: In GitHub Desktop, commit the edited file with a descriptive message (such as, "Initial commit: Update my course homepage text") and push the changes to GitHub.
  • Check Your Live URL: Wait about two minutes. Your live homepage URL is the one you saw when setting up your course portfolio. Your homepage should now be live on the web.

Exercise Submission Details

Your submitted URL must be live and publicly accessible, and the page must display your name and any other personalized information you have added.
